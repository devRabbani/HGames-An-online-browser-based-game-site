About

It is an online-based HTML5 game site, where you can play the games without the need to download.It has newly updated PWA features so that you can install it on your phone or PC and play offline. This app uses some special cache storing and clearing strategy so that you can use your app offline as well and every three days this cache will be cleared automatically.It is built with some cool JavaScript features. It is created by CanWeBe! With the help of Html, CSS and JavaScript with the addition of Workbox libraries.The games are crawled from internet in a json format.We update our games list everyday, now enjoy our app
